# Glossary

## A

### Action
In ROS 2, a goal-oriented communication pattern that provides feedback and can be canceled.

### AI (Artificial Intelligence)
The simulation of human intelligence processes by machines, especially computer systems.

## B

### Bipedal Movement
Locomotion using two legs, a key capability for humanoid robots.

## C

### Cognitive Architecture
A framework for how intelligent agents work, including perception, reasoning, and action.

## D

### Digital Twin
A virtual representation of a physical object or system that can be used for simulation and analysis.

## E

### Environment
The physical or simulated space in which a robot operates.

## F

### Forward Kinematics
The process of determining the position of the robot's end-effector given joint angles.

## G

### Gazebo
A physics-based simulation environment for robotics.

## H

### Hardware Abstraction Layer (HAL)
Software layer that allows the same code to run on different hardware platforms.

## I

### Isaac ROS
NVIDIA's collection of perception and manipulation packages for ROS 2.

### Isaac Sim
NVIDIA's robotics simulator for developing and testing AI-based robotics applications.

## J

### Joint State
The position, velocity, and effort of a robot's joints.

## K

### Kinematics
The study of motion without considering the forces that cause it.

## L

### LLM (Large Language Model)
A type of artificial intelligence model designed to understand and generate human language.

## M

### Module
A self-contained unit of the course curriculum focusing on specific topics.

## N

### Nav2
The navigation stack for ROS 2 that provides path planning and execution capabilities.

### Node
A process that performs computation in ROS.

## O

### Odometry
The use of data from motion sensors to estimate change in position over time.

## P

### Perception
The ability of a robot to understand its environment through sensors.

### Publisher
A ROS node that sends messages to a topic.

## R

### Robot Operating System (ROS)
A flexible framework for writing robot software.

### ROS 2
The second generation of the Robot Operating System with improved architecture.

## S

### Sensor Fusion
The process of combining data from multiple sensors to improve accuracy and reliability.

### Service
A synchronous request/response communication pattern in ROS.

### Simulation
The imitation of the operation of a real-world process or system over time.

## T

### Topic
A named bus over which ROS nodes exchange messages.

### Transform (TF)
The relationship between coordinate frames in a robot system.

## U

### URDF (Unified Robot Description Format)
An XML format for representing a robot model in ROS.

## V

### VSLAM (Visual Simultaneous Localization and Mapping)
The process of calculating the position and mapping the environment using visual input.

### Vision-Language-Action (VLA)
A system that integrates visual perception, language understanding, and physical action.

## W

### Whisper
OpenAI's speech recognition system.

## X, Y, Z

### Xacro
An XML macro language for generating URDF files.
